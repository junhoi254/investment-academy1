## 💡 추가 메시지 넣기

### 나중에 더 추가하려면:

#### 방법 1: 관리자 페이지에서 수동 입력
```
1. 관리자 로그인
2. 무료 채팅방 입장
3. 메시지 직접 입력
```

#### 방법 2: Python 스크립트로 추가
```python
import sqlite3
from datetime import datetime

conn = sqlite3.connect('trading_chat.db')
cursor = conn.cursor()

# 메시지 추가
cursor.execute('''
    INSERT INTO messages (room_id, user_id, content, message_type, created_at)
    VALUES (?, ?, ?, ?, ?)
''', (1, 1, "새 메시지 내용", 'text', datetime.now().isoformat()))

conn.commit()
conn.close()
```
