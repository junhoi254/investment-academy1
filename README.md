# 🎓 투자학당 - Investment Academy

AWS 무료 티어로 배포 가능한 실시간 트레이딩 리딩방 + 모바일 앱

## 📋 주요 기능

### 1️⃣ 인증 및 권한 관리
- ✅ 전화번호 기반 회원가입 (자동 - 추가)
- ✅ 관리자 승인 후 로그인
- ✅ 4단계 권한: **일타훈장님(관리자)** / **서브관리자** / 직원 / 회원
- ✅ 회원 기간 관리 (만료일 설정)

### 2️⃣ 채팅방
- ✅ 무료 채팅방: 누구나 열람 가능, 일타훈장님/서브관리자만 메시지 작성
- ✅ 유료 채팅방: 로그인 후 이용 가능, 일타훈장님/서브관리자만 메시지 작성
- ✅ 실시간 WebSocket 채팅
- ✅ 리딩방 타입: 주식 / 해외선물 / 코인선물
- ✅ **파일 업로드** (PDF, DOC, XLSX 등)
- ✅ **이미지 업로드** (JPG, PNG 등)
- ✅ **이모티콘** (100+ 이모티콘)

### 3️⃣ 관리자 기능 (일타훈장님)
- ✅ 회원/서브관리자 관리
- ✅ 회원 기간 설정/연장
- ✅ 서브관리자 계정 생성
- ✅ 채팅방 생성/관리

### 4️⃣ 특수 기능
- ✅ 뉴스 크롤링
- ✅ MT4 포지션 자동 전송
- ✅ 실시간 시그널 알림

### 5️⃣ 모바일 앱
- ✅ iOS / Android 지원

## 🎯 역할별 권한

- **지원**: PDF, DOC, DOCX, XLS, XLSX, TXT, ZIP

- **개수**: 100+
- **방법**: 😊 버튼
