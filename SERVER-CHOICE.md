## 🔄 서비스 간 이동

### Replit → Render 이동:
```
1. Replit에서 잘 작동하는지 확인
2. GitHub에 코드 업로드
3. Render에서 배포
4. URL 변경 공지

소요 시간: 30분
```

### Render → Railway 이동:
```
1. GitHub 코드 그대로 사용
2. Railway에서 저장소 연결
3. 자동 배포
4. URL 변경 공지

소요 시간: 20분
```

### 무료 → AWS 이동:
```
1. EC2 인스턴스 생성
2. 배포 스크립트 실행
3. 도메인 연결
4. 안정적 운영

소요 시간: 2시간
```

## ⚡ 슬립 모드 해결 방법

### 문제:
Replit, Render는 미사용 시 자동 종료

### 해결책: UptimeRobot (무료)

```
1. https://uptimerobot.com 가입
2. 모니터 추가
3. 5분마다 자동 ping
4. 서버 항상 깨어있음!
```

**설정 시간:** 5분  
**효과:** 24시간 실행 유지
