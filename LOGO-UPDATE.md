## 🎨 디자인 특징

### 로고 스타일
- **크기**: 150x150px (로그인), 120x120px (회원가입)
- **모양**: 원형 (border-radius: 50%)
- **테두리**: 보라색 (#667eea) 테두리
- **효과**: 
  - 그림자 효과
  - 호버 시 확대 (scale 1.05)
  - 그림자 강화

### 색상 테마
- **주 색상**: #667eea (보라색)
- **부제목**: 보라색 텍스트
- **배경**: 흰색 카드

## 🚀 적용 방법

프로젝트를 다시 다운로드하면 자동으로 적용됩니다!

```bash
# 압축 해제
tar -xzf investment-academy.tar.gz
cd investment-academy

# 실행
./start.sh
```

브라우저에서 http://localhost:3000 접속하면  
**일타훈장님 사진이 로그인 화면에 나타납니다!** 🎉

## 📱 모바일 앱
